# Flipkart_clone
A frontend clone of E-commerce website of Flipkart. This is created using HTML and CSS. The page is created for big screens i.e, laptops, PCs but can also work on mobile as well.

# Technologies:
* HTML5
* CSS3
* CSS Animations
* Netlify
